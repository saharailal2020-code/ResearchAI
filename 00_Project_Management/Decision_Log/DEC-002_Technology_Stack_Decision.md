# DEC-002

## Judul

Menetapkan Technology Stack awal untuk ResearchAI.

---

## Tanggal

8 Juli 2026

---

## Keputusan

ResearchAI akan menggunakan stack teknologi awal sebagai berikut:

- Backend: Python + FastAPI
- Database: PostgreSQL
- Frontend: React
- Styling: Tailwind CSS
- AI: OpenAI API
- File Storage: Local storage pada tahap awal
- Deployment: Docker
- Version Control: Git + GitHub

---

## Alasan

Stack ini dipilih karena:

- Cocok untuk pengembangan aplikasi berbasis data dan AI.
- Python kuat untuk backend, data processing, dan integrasi AI.
- FastAPI ringan, modern, dan cocok untuk API.
- PostgreSQL stabil untuk data relasional proyek riset.
- React cocok untuk membangun dashboard dan aplikasi operasional.
- Tailwind CSS mempercepat pembuatan UI yang konsisten.
- Docker memudahkan deployment dan setup environment.
- GitHub sudah digunakan sebagai repository ResearchAI.

---

## Dampak

Seluruh desain teknis berikutnya akan mengacu pada stack ini, termasuk:

- System Architecture
- Database Schema
- API Requirement
- Backend Development
- Frontend Development
- Deployment Setup
